NineML

Introduction

NineML is a family of Invisible XML tools.

  • CoffeeGrinder implements both an Earley parser and a GLL parser. This parser forms the basis for the Invisible XML parser, but is limited to simple grammars. The JavaDoc describes the API.

  • CoffeeFilter is an implementation of an Invisible XML processor. This project provides the underlying Invisible XML parsing. The JavaDoc describes the API.

  • CoffeePot is a command line tool for parsing Invisible XML grammars and transforming inputs with them. Consult the user’s guide for more information.

  • CoffeeSacks provides a set of Saxon extension functions for processing Invisible XML documents from with XSLT or XQuery. Consult the user’s guide for more information.

  • CoffeePress is an XProc 3.0 step for XML Calabash.
